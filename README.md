# UTN-FRA Laboratorio III

Servidor node.js con BD MySQL y autentificacion JWT

## Instrucciones

Instalar las dependencias:

``` bash
$ npm install
```

Iniciar el servidor:

``` bash 
$ node app.js
```

Importar la tabla usuarios.

Puede usar la carpeta ./public como servidor para una aplicacion web.
